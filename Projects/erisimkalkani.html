AccessShield

A security application designed for people with disabilities. Accessible, intelligent, and human-first.

Key Features

  • 🔊 Voice-based security alerts
  • 👓 Fully screen-reader compatible
  • 🔍 AI-based suspicious app scanning
  • 🛑 Offline functionality — works without internet
  • 📱 One-tap emergency privacy mode
  • 🧠 Adaptive layout for motor disabilities
